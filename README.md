# WordPress Blocks Live Tester

A modern web application for browsing and testing WordPress block plugins directly in your browser. Powered by the [WordPress Playground](https://playground.wordpress.net/), it allows you to test blocks in a real, ephemeral WordPress environment with zero configuration and no local server required.

🌐 **Live Demo:** [wp-blocks.com](https://wp-blocks.com)

## ✨ Features

- **Live Search:** Quickly find WordPress block plugins using the official WordPress.org API.
- **Instant Preview:** Test blocks immediately via the WordPress Playground integration.
- **Infinite Scroll:** Seamlessly browse through hundreds of plugins.
- **Masonry Layout:** Responsive, optimized grid display for plugin cards.
- **Client-Side Only:** Fast, secure, and entirely browser-based execution.

## 🛠️ Tech Stack

- **[Vite](https://vitejs.dev/)** - Next-generation frontend tooling
- **TypeScript** - Strict typed logic for robust development
- **[MiniMasonry.js](https://spope.github.io/MiniMasonry.js/)** - Lightweight responsive grid layout
- **WordPress Playground API** - In-browser PHP and WordPress execution
